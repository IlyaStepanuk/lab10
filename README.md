# Лабораторная работа по Git и Markdown

## 1. Форматирование текста
- **Полужирный текст** (`**текст**` или `__текст__`)
- *Курсив* (`*текст*` или `_текст_`)
- ~~Зачёркнутый текст~~ (`~~текст~~`)
- `Моноширинный текст` (обратные кавычки)

## 2. Списки

### Ненумерованный список
- Первый элемент
- Второй элемент
  - Вложенный элемент
  - Ещё один вложенный
- Третий элемент

### Нумерованный список
1. Первый пункт
2. Второй пункт
   1. Вложенный пункт
   2. Ещё один
3. Третий пункт

## 3. Подзаголовки разного уровня
# Заголовок 1 уровня

### Заголовок 3 уровня
#### Заголовок 4 уровня
##### Заголовок 5 уровня
###### Заголовок 6 уровня
